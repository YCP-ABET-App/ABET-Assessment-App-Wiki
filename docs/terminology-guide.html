Terminology Guide

Understanding ABET Assessment

ABET assessments provide accredited institutions with a structured mechanism to measure student learning outcomes, evaluate program effectiveness, and monitor long-term curricular improvement. These assessments also serve as formal evidence to ABET that the institution's engineering and technology programs continue to meet established accreditation criteria and program objectives.

ABET's goals in this context are formally referred to as "Student Outcomes". To properly describe the assessment process, it is necessary to understand several related terms that are central to ABET's continuous improvement framework.

Core Terminology

Student Outcome

Definition: What students are expected to know and be able to do by the time of graduation. These relate to the knowledge, skills, and behaviors that students acquire as they progress through the program.

Example: "Students will be able to design and implement a complete software system using industry-standard practices."

Assessment

Definition: One or more processes that identify, collect, and prepare data to evaluate the attainment of student outcomes. Effective assessment uses relevant direct, indirect, quantitative and qualitative measures as appropriate to the outcome being measured. Appropriate sampling methods may be used as part of an assessment process.

Example: Administering a comprehensive exam, analyzing student projects, or reviewing portfolios of completed work.

Evaluation

Definition: One or more processes for interpreting the data and evidence accumulated through assessment processes. Evaluation determines the extent to which student outcomes are being attained. Evaluation results in decisions and actions regarding program improvement.

Example: Faculty reviewing exam results to determine if 80% of students achieved the desired competency level.

Performance Indicator

Definition: A type of assessment utilized by many accredited institutions, including York College of Pennsylvania. Performance indicators correspond to individual student outcomes. Performance indicators are usually defined by an abstract goal.

Example: "Students can correctly interpret a computational problem and define its parameters."

Note: Each Student Outcome can contain many corresponding performance indicators.

Measure

Definition: A type of assessment used to provide data on performance indicators. Measures are typically defined by a specific task or observation.

Example: A specific question on an exam or a homework assignment; performance on an individual exam question.

Note: Each performance indicator may be linked to several different measures.

FCAR (Faculty Course Assessment Report)

Definition: An acronym which stands for Faculty Course Assessment Report. In the context of the ABET App, an FCAR is a completed measure that has been properly formatted.

Purpose: FCARs provide instructors with a standardized way to submit assessment data to program administrators.

Recommended Action

Definition: A set of actions that an institution will take going forward to improve the effectiveness of their program by better achieving student outcomes.

Example: "Revise the curriculum to include more hands-on laboratory work" or "Implement additional tutoring support for struggling students."

The Assessment Process at York College of Pennsylvania

At YCP, assessments are performed every semester based on an assessment schedule. Here's how the process works:

Step-by-Step Process

  1. Course Mapping: Based on a course mapping, each course scheduled for assessment is assigned corresponding performance indicators, and by extension, corresponding student outcomes.
  2. Performance Indicator Assessment: Each performance indicator requires the instructor(s) to conduct one or more measures.
  3. Data Collection: The data for each measure could be determined by:
    • Instructor observations
    • Performance on assignments
    • Performance on specific parts of assignments (e.g., performance on an individual exam question)
  4. FCAR Submission: Each measure is formatted by the instructor into an FCAR and submitted to a program administrator.
  5. Data Storage: Administrators ensure that every FCAR is securely stored for future use.
  6. Annual Assessment Retreat: At the end of each year, faculty meet to:
    • Organize assessment data into tables called data summaries
    • Arrange data first by student outcomes, then by performance indicators
    • Perform evaluation of the data to determine the degree to which each student outcome has been achieved
    • Identify recommended actions to improve outcomes
    • Determine if changes should be made to the performance indicators

The Six-Year Accreditation Cycle

Every six years marks the ABET Accreditation Cycle, which is a critical milestone for accredited institutions.

What Happens During the Cycle

Data from the previous six annual retreats must be reorganized and presented to ABET. The data summaries maintain a specific structure:

  • Data is organized according to student outcome above all else
  • The first data summary contains the data for the first student outcome over the course of six years
  • Each student outcome data summary includes a narrative on all recommended actions pertaining to that outcome taken throughout the six-year cycle
  • This pattern repeats for every student outcome

Why This Matters

This is a complicated process that requires careful planning and coordination amongst the faculty, as well as the secure storage of data over a period of many years. The data must be precise, well-organized, and easily accessible by ABET reviewers.

Why the ABET App Exists

The ABET Assessment App was created to lessen the burden of the assessment process on faculty by automating and simplifying key tasks. The application accomplishes the following:

✓ Secure, Centralized Storage

Store all assessment data in a secure, centrally accessible location, eliminating the need for scattered files and spreadsheets.

✓ Simplified Instructor Submission

Simplify the submission process of measure data for instructors through an intuitive web application interface, reducing the time and effort required to format and submit FCARs.

✓ Streamlined Administrator Dashboard

Simplify the collection process of measure data for administrators via a comprehensive dashboard that provides visibility into all submitted assessments and their status.

✓ Automatic Data Summary Generation

Automatically format assessment data into properly organized data summaries, saving countless hours during annual retreats and the six-year accreditation cycle.

By centralizing data management and automating routine tasks, the ABET App allows faculty to focus on what matters most: interpreting assessment results and improving student learning outcomes.
