Assessment Management

Creating Assessments

Assessments are the core of the application. Each assessment tracks outcomes and evaluations for a specific program and year.

To create a new assessment:

  1. Click on "Assessments" in the main menu
  2. Click "Create New Assessment"
  3. Fill in the following information:
    • Program Name
    • Academic Year
    • Assessment Period
    • Description
  4. Click "Save"

Editing Assessments

You can modify assessment details at any time before the assessment is closed:

  1. Select the assessment from the list
  2. Click "Edit"
  3. Make your changes
  4. Click "Save"

Viewing Assessment Status

The assessment status page shows:

  • Current assessment stage
  • Progress completion percentage
  • Assigned evaluators
  • Number of outcomes entered
  • Evaluation completion status

Data Entry

Adding Program Outcomes

Program outcomes are the goals your program aims to achieve. To add outcomes:

  1. Navigate to the assessment
  2. Click "Outcomes" tab
  3. Click "Add Outcome"
  4. Enter the outcome description
  5. Select the outcome type
  6. Click "Save"

Entering Student Performance Data

Record how students performed against the defined outcomes:

  1. Click on "Student Data" tab
  2. Click "Import Data" or "Manual Entry"
  3. For manual entry:
    • Enter student ID
    • Select the relevant outcome
    • Rate performance (1-5 scale)
    • Add any notes
  4. Click "Save"

Managing Rubrics

Rubrics provide evaluation standards. To create a rubric:

  1. Go to "Rubrics" section
  2. Click "Create New Rubric"
  3. Define rating levels and descriptions
  4. Add evaluation criteria
  5. Save the rubric

Reporting

Generating Reports

The application provides several report types:

  • Summary Report: Overview of all outcomes and evaluations
  • Detailed Report: Complete data with all entries
  • Trend Report: Comparison across multiple years
  • Export Report: Download data in Excel or PDF format

Exporting Data

To export assessment data:

  1. Open the assessment
  2. Click "Export" button
  3. Select format (Excel or PDF)
  4. Click "Download"

Troubleshooting

Common Issues

  • Cannot login: Verify your username and password. Contact administrator for account reset.
  • Data not saving: Check your internet connection and try again. Ensure all required fields are filled.
  • Slow performance: Try clearing your browser cache or using a different browser.
  • Export failed: Try a different export format or reduce the data range.
