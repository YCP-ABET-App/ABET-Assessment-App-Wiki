Getting Started

Introduction

Welcome to the ABET Assessment Application! This guide will help you get up and running with the system in just a few minutes.

Prerequisites

  • A valid user account (provided by your institution)
  • A modern web browser (Chrome, Firefox, Safari, or Edge)
  • Stable internet connection

Step 1: Login

Navigate to the application home page and enter your credentials. You will receive login information from your system administrator.

  1. Open the application URL in your browser
  2. Enter your username
  3. Enter your password
  4. Click "Login"

Step 2: Navigate the Dashboard

After logging in, you'll see the main dashboard. The dashboard contains:

  • Navigation Menu: Quick access to main features
  • Recent Items: Quick links to recently accessed assessments
  • Status Overview: Current assessment status at a glance

Step 3: Create Your First Assessment

To create a new assessment:

  1. Click "New Assessment" on the dashboard
  2. Fill in the assessment details
  3. Select the relevant program and year
  4. Click "Create"

Step 4: Add Assessment Data

Once your assessment is created, you can:

  • Add assessment criteria
  • Import student data
  • Create evaluation rubrics
  • Assign evaluators

Getting Help

If you encounter any issues, refer to:

  • The User Guide for detailed instructions
  • The FAQ section for common questions
  • Contact your system administrator for account issues
