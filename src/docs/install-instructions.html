Getting Started

Introduction

Welcome to the ABET Assessment Application! This guide will help you get up and running with the system in just a few minutes.

Prerequisites

  • A Digital Ocean droplet with at least 2GB of memory and 50GB of storage
  • A GitHub account

Step 1: Setup Droplet

Navigate to Digital Ocean's website and click 'login' in the top right.

Once logged in, we now should be on the projects page and now we'll create the droplet:

  1. Click on 'Droplets' on the left hand side
  2. Then click 'Create Droplet' in the top right
  3. Select a region and datacenter that is closest to where your userbase will be
  4. For operating system, we'll choose Ubuntu 25.04
  5. For the plan, the minimum that we recommend is the $12/mo plan which has 2GB of memory and 50GB of storage
  6. Add additional storage or automatic backups if you like
  7. We can skip over the SSH keys for now
  8. Give the droplet a name and tags if you choose, and then hit 'Create Droplet'

Step 2: Setting up the firewall

Once your droplet is created, we'll now need to add firewall rules so your institution can connect to the application:

  • In the administrator panel of you droplet on Digital Ocean's website, click the 'Networking' button
  • Here, scroll down to the section 'Firewalls', here we'll need to add some special rules
  • Click button under the 'Firewalls' header to open the firewall rules
  • The resulting page may not immediately show all the rules, if so navigate to the 'droplets' tab then back to 'rules'
  • Here, we need to ensure that we have five Inbound rules:
    1. SSH: TCP port 22, source: your IP address (for server management)
    2. HTTPS: TCP port 443, source: All IPv4 and IPv6
    3. Custom: TCP port 3000, source: All IPv4 and IPv6 (access to the web application through the browser)
    4. Custom: TCP port 8080, source: All IPv4 and IPv6
    5. Custom: TCP port 8081, source: All IPv4 and IPv6
  • Once these are created, now we can move on to adding the dependencies in our droplet.

Step 3: Downloading the project

Now that we've added our firewall rules, we can go to the 'Access' tab and launch our droplet's console:

  1. Click "New Assessment" on the dashboard
  2. Fill in the assessment details
  3. Select the relevant program and year
  4. Click "Create"

Step 4: Add Assessment Data

Once your assessment is created, you can:

  • Add assessment criteria
  • Import student data
  • Create evaluation rubrics
  • Assign evaluators

Getting Help
